建站

首先得确保电脑中有node.js 官网 https://nodejs.org/zh-cn

npm install -g hexo-cli

安装 Hexo 完成后，请执行下列命令，Hexo 将会在指定文件夹中新建所需要的文件

生成静态页面

hexo g

本地调试

hexo s

修改主题

此时hexo使用的是默认主题landscape，使用本站主题(3-hexo主题)请参考 https://github.com/yelog/hexo-theme-3-hexo

git clone https://github.com/yelog/hexo-theme-3-hexo.git themes/3-hexo

修改hexo根目录的_config.yml

theme: 3-hexo

部署

部署在白嫖服务器

  • 静态页面上传在github仓库中的可尝试vercel部署
  • 静态页面上传在gitee仓库中可参照此视频直接部署

部署在自己服务器

方式一：手动把.deploy_git下生成的静态文件复制到自己的服务器上

方式二：

1.修改hexo根目录的_config.yml中deploy值

git@github.com:mohoho/hexo-theme-3.git 需要在github中提前建好仓库

2.部署到github

npm install hexo-deployer-git --save

hexo d

3.在部署目录使用git
git clone git@github.com:mohoho/hexo-theme-3.git
